# クラスター情報

フォーマットは以下の通りです

```json
{
  // cluster identifier
  "main": {
    // same here
    "identifier": "main",
    // pmmp server address
    "address": "0.0.0.0",
    // pmmp server port
    "port": 19132,
    // address for ipc
    "ipc_address": "127.0.0.1",
    // port for ipc
    "ipc_port": 21010
  }
  // ...
}
```
